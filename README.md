# 鞋业开单管理系统 (Shoe Billing System) V1.1.2

这是一个专为鞋业销售设计的开单与报价管理系统，支持商品管理、库存管理、报价单生成、销售开单及历史记录查询。

## 更新日志 (V1.1.2)

*   **功能优化与增强**：
    *   **默认语言**：所有打印单据默认设置为中文 (zh)。
    *   **新增排序**：商品列表现在按新增时间降序排列（最新添加的商品排在最前）。
    *   **分享增强**：报价单、备货单及销售单新增“复制为图片”功能，方便一键分享到微信/QQ。
    *   **UI 升级**：统一了报价单、备货单与销售单的视觉风格，界面更加专业美观。
    *   **安全确认**：新增商品成功后增加提示，删除商品操作增加二次确认弹窗。
    *   **快捷键**：全局支持 `Del` 键快速删除选中项。
*   **版本 1.1.1 历史更新内容**：
    *   新增库存管理功能（入库、出库、散数/整箱模式）。
    *   新增出入库流水记录。

## 功能特性

*   **商品管理**：轻松添加、编辑和删除商品信息，支持图片上传。
    *   包含字段：货号、标签、单价、最小起订量(箱)、成本价、码段、颜色、每箱数量、**库存数量**。
*   **库存管理**：
    *   实时库存监控。
    *   便捷的入库与出库操作。
    *   详细的库存变动流水记录。
*   **报价与开单**：
    *   快速生成报价单（Quote），支持网格/列表/模板多种模式。
    *   将报价单转换为正式销售单（Bill）。
    *   自动计算金额与总箱数。
*   **记录查询**：
    *   查看历史报价和开单记录。
    *   支持编辑后重新打印。
*   **打印功能**：支持生成单据图片并打印。
*   **数据安全**：本地数据存储（JSON格式），支持激活码验证。

## 如何使用

### 方式一：直接运行可执行文件
在 `dist` 目录下找到 `ShoeBillingApp-S.exe` 双击运行即可。
*注意：首次运行可能需要输入激活码。*

### 方式二：源码运行
需要安装 Python 3.x 及以下依赖库：
```bash
pip install pillow
```
运行命令：
```bash
python ShoeBillingApp-S.py
```

## 目录结构说明

*   `ShoeBillingApp-S.py`: 主程序源代码
*   `dist/`: 包含打包好的可执行文件
*   `products.json`: 商品数据库
*   `inventory_history.json`: 库存变动历史记录
*   `billing_history.json`: 开单历史记录
*   `quote_history.json`: 报价历史记录
*   `images/`: 存放商品图片（如有）
